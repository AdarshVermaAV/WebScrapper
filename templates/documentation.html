Scraper API Documentation

Overview

The Scraper API is a web scraping tool built using Flask and Selenium. It allows users to scrape data from provided URLs and extract various elements like paragraphs, headings, images, and more, including content from social media platforms.

Endpoints

/scrape

This endpoint accepts a POST request with a JSON payload containing URLs to scrape.

Request Format

{
    "urls": ["http://example.com", "http://another-example.com"]
}

Response Format

{
    "http://example.com": {
        "paragraphs": "Extracted paragraph text...",
        "urls": ["http://referral1.com", "http://referral2.com"],
        "headings": ["Heading 1", "Heading 2"],
        "images": ["http://image1.com", "http://image2.com"],
        "list_items": ["Item 1", "Item 2"],
        "span_texts": ["Span text 1", "Span text 2"],
        "table_data": [["Row 1 Col 1", "Row 1 Col 2"], ["Row 2 Col 1", "Row 2 Col 2"]]
    },
    "http://another-example.com": {
        "error": "Error message..."
    }
}

Usage

  1. Run the Flask application by executing python app.py.
  2. Make a POST request to http://127.0.0.1:5000/scrape with the required JSON payload.
  3. Receive a JSON response with the scraped data.

Key Features

  • Extracts paragraphs, URLs, headings, images, list items, span texts, and table data.
  • Filters referral URLs to include only those from snipnutrition.com.
  • Handles errors gracefully and returns informative messages.
  • Supports scraping from social media sites like Facebook, Instagram, and Twitter.
